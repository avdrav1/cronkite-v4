## Summary

Build an Add Feed modal with two approaches:

**Browse Tab:**
- Searchable, filterable grid of curated feeds
- Category pills for quick filtering
- One-click add with quick folder/priority popover
- Visual feedback for already-added feeds

**Custom URL Tab:**
- URL input with real-time validation
- Auto-detect feeds from website URLs
- Rich preview (name, description, recent articles)
- Folder and priority configuration
- Create new folder inline

Both flows should feel fast, give clear feedback, and handle errors gracefully.
