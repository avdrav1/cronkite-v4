# Cronkite Onboarding Flow â€” Replit Agent Prompt

Build a **premium, animated onboarding wizard** for an RSS reader app called "Cronkite". This is the first-time user experience that personalizes their feed.

---

## Overview

A **5-step wizard** that:
1. Welcomes the user
2. Lets them pick interests (categories)
3. Optionally adds local news (region)
4. Shows recommended feeds based on selections
5. Confirms and launches the reader

**This must feel premium** â€” smooth animations, beautiful typography, not a generic form wizard.
